### `backend/tests/runtime/test_w2_3_r4_integration.py` (NEW)

1. **`test_all_five_w23_layers_coherent_across_five_turns`**
   - Executes 5 real turns with varied triggers (including character-named triggers)
   - Asserts all five W2.3 layers are present, have correct types, and contain turn-specific data
   - Verifies boundedness: ProgressionSummary fields limited, salient_axes ≤ 10, selected_units ≤ 15
   - Proves layer distinctness: all different types, different content
   - **The canonical snapshot proof**

2. **`test_w23_layer_state_changes_are_detectable_across_turns`**
   - Snapshots state after turn 1 (history_size, progression_turns, short_term_context.turn, relationship_turn)
   - Continues execution to turn 5
   - Asserts all values measurably changed (increased)
   - Proves layers are not static (actually being updated on each turn)
   - **Fails if accumulation/derivation wiring is removed**

3. **`test_w23_history_trimming_leaves_derived_layers_coherent`**
   - Pre-configures SessionHistory(max_size=3) to force FIFO trimming
   - Executes 5 turns (history trims turns 1-2, keeps 3-5)
   - Asserts history correctly trimmed: size==3, entries[0].turn_number==3
   - Asserts ProgressionSummary reflects trimmed history: total_turns_in_source==3
   - Verifies all five layers still present and coherent after trimming
   - Verifies boundedness maintained: selected_units ≤ 15, salient_axes ≤ 10
   - **Proves FIFO trimming doesn't break coherence**

**Summary**: 3 strong tests, 100% passing, comprehensive integration coverage
