# 06 — World Engine Product Specification

The world-engine is the authoritative runtime control and inspection product surface for story authors, live operators, QA engineers, and narrative systems engineers.

## Product areas
- Live runtime control
- Authoring studio
- Diagnostics
- QA and replay

## Live runtime control includes
- session list
- current scene
- story-health dashboard
- active strategy
- degraded mode visibility
- safe event injection
- NPC behavior overrides
- bounded branch steering
