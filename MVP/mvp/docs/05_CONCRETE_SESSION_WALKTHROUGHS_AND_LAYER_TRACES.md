# 05 — Concrete Session Walkthroughs and Layer Traces

## Turn 0 — Opening

Player-visible output:
“You have been hired to look into Vera Chen. Missing money. A nervous client. Too many things left unsaid.”

Layer trace:
- Consciousness: `establish_mystery`
- Emotional: keep cognitive load low, preserve agency
- Narrative: opening beat, low-to-moderate tension, plant future money trail
- Semantic: canonical opening commit
- Data: opening written to episodic memory and canonical session state

## Turn 6 — Player is confused

Input:
“I’m confused about the timeline.”

Result:
- Emotional detects confusion and support need
- Consciousness allows a support move without dropping mystery integrity
- Narrative converts recap into a dramatic recap
- Semantic emits `CONSTRAINT_RESPECTING_REDIRECT`

Output:
