# 25 — Memory Interaction Laws

1. Truth stays truth.
2. Effect is not equal to truth.
3. Belief is carrier-bound.
4. Myths require densification.
5. Suppression is active.
6. Sacred memory has stronger merge, edit, and access protection.
7. Counter-memory may challenge official memory without auto-becoming truth.
8. Ontological memory requires strong thresholds.
