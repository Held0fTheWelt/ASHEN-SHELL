# 21 — Governed Consolidation Pipeline Specification

Pipeline:
raw episodic material → duplicate handling → clustering → stable pattern extraction → provisional semanticization → governed promotion candidates → approved semantic or procedural memory

Protections:
- low-confidence material stays low
- consolidation is blocked on unresolved high-severity conflicts
- exact duplicates, near duplicates, and semantic siblings are treated differently
- procedural promotion requires stricter evidence than semantic promotion
