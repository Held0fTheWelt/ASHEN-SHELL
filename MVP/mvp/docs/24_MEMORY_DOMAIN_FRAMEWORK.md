# 24 — Memory Domain Framework

Core domains:
- canonical truth
- episodic experience
- belief
- social
- institutional
- cultural
- rumor / memetic
- legend / mythic
- sacred / ritual
- trauma / suppression
- counter-memory / research
- ontological / reality-bearing
- procedural

Every domain record carries:
- domain type
- carrier scope
- authority class
- source lineage
- temporal validity
- access rules
- conflict rules
- promotion path
- effect surface
